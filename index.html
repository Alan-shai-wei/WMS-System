作業控制塔（Control Tower）

效率排名邏輯：
1. 一般配效率 = 一般配總量 ÷ 撿貨工時。
2. 無包裝效率 = 無包裝總PCS ÷ 撿貨工時。
3. 若同一員工今天兩種作業都有，依主要作業類型判定效率值。
4. 若撿貨工時為 0，效率值以 0 計算。
5. 目前只顯示今日前 5 名。

貨態分流（Flow Control）

新版貨態流程：
一般配：待撿貨 → 撿貨中 → 待配貨 → 配貨中 → 待包貨 → 已完成
無包裝：待撿貨 → 撿貨中 → 待包貨 → 包貨中 → 已完成

請先選擇上方類型（一般配 / 無包裝），再選擇下方流程狀態查看批次
